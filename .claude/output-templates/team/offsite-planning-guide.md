## Variant A: Strategic Offsite

### Half-Day (4 hours)

| Time | Session | Output | Facilitation note |
|------|---------|--------|-------------------|
| 0:00-0:15 | **Opening and ground rules** | Objectives posted, norms set | Phones away. Post "will not discuss" list visibly. |
| 0:15-0:45 | **Context setting** | Shared fact base | No debate -- questions only. Data pack sent in pre-work. |
| 0:45-1:45 | **Working session 1:** [Topic] | [E.g., "Strategic options generated and clustered"] | Start with 5 min silent individual writing before group discussion. |
| 1:45-2:00 | **Break** | -- | Fifteen minutes. No working through it. |
| 2:00-3:00 | **Working session 2:** [Topic] | [E.g., "Options force-ranked against criteria"] | Use dot-voting if the group is stuck. |
| 3:00-3:30 | **Decision capture** | Decisions separated from open questions | Read aloud: "Does anyone disagree as stated?" Silence is consent. |
| 3:30-4:00 | **Commitments and close** | Owners and deadlines assigned | Use Follow-Up Template below. |

### Full-Day (8 hours)

| Time | Session | Output | Facilitation note |
|------|---------|--------|-------------------|
| 0:00-0:20 | **Opening** | Norms and objectives | -- |
| 0:20-1:00 | **Context setting** | Shared facts | Q&A only. |
| 1:00-2:15 | **Working session 1:** [Topic] | [Specific output] | Alternate individual, small-group, and plenary. |
| 2:15-2:30 | **Break** | -- | -- |
| 2:30-3:45 | **Working session 2:** [Topic] | [Specific output] | Assign a devil's advocate per option. |
| 3:45-4:30 | **Lunch** | Unstructured | Seat people next to those they work with least. |
| 4:30-5:45 | **Working session 3:** [Topic] | [Specific output] | Hard trade-offs happen here. Do not rush. |
| 5:45-6:30 | **Decision capture** | Locked decisions | Write and read aloud. |
| 6:30-7:15 | **Commitments and close** | Owners, deadlines, retro on offsite | What worked? What would we change? |

## Variant B: Team-Bonding Offsite (Half-Day, 4 hours)

| Time | Session | Output | Facilitation note |
|------|---------|--------|-------------------|
| 0:00-0:15 | **Opening** | Safety norms established | "Nothing shared here becomes ammunition later." Leader states this explicitly. |
| 0:15-1:00 | **Personal histories** | Empathy through backstory | Each person: where they grew up, siblings, one unusual childhood challenge. Leader goes first, sets the depth. |
| 1:00-1:45 | **Strengths and contributions** | Direct appreciation | Written first, read aloud: "The thing you do that makes this team better is ___." |
| 1:45-2:00 | **Break** | -- | -- |
| 2:00-2:45 | **Team narrative** | Shared story | Three chapters: where we started, hardest moment, what we are building. Use Appreciative Inquiry. |
| 2:45-3:30 | **Shared experience** | Memory created together | Walk, cook, build, play. The activity matters less than the shared memory. |
| 3:30-4:00 | **Commitments to each other** | Relational commitments | Not deliverables -- e.g., "I commit to giving feedback within 24 hours." |

## Required Materials

Printed objectives and exclusion list (posted on walls), large sticky notes (1 pad per person), thick markers (1 per person), visible timer, 2-3 whiteboards or flip charts, dot stickers for voting (10 per person), camera or designated note-taker for whiteboard capture.
